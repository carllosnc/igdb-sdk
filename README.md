## Usage

```bash
bun install
bun run index.ts
```

## Authentication

Requires a Twitch Client ID and Client Secret via `.env`:

```
TWITCH_CLIENT_ID=your_client_id
TWITCH_CLIENT_SECRET=your_client_secret
```
